Introduction to HTML and Creating the Document Structure
Question 1: What is HTML?
Answer: HTML stands for HyperText Markup Language. It is the standard markup language used to create the structure and presentation of web pages.

Question 2: What is the purpose of the <!DOCTYPE> declaration in HTML?
Answer: The <!DOCTYPE> declaration specifies the HTML version being used in the document. It helps the browser to understand how to interpret the HTML code.

Question 3: What are the basic HTML tags for document structure?
Answer: The basic HTML tags for document structure include <html>, <head>, <title>, and <body>.

Question : How do you create a hyperlink in HTML?
Answer: Hyperlinks in HTML are created using the <a> tag. The href attribute specifies the URL of the destination page.

Question 5: What is the purpose of the <img> tag in HTML?
Answer: The <img> tag is used to embed images in HTML documents. The src attribute specifies the source (URL or file path) of the image.
